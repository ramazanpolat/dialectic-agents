# Axiom: Law of Identity

## Statement
A thing is identical to itself. (A = A)

## Justification
Denial produces immediate incoherence. If A ≠ A, no statement can refer to anything, no predicate can attach to any subject, and language itself collapses. This is not a claim about the world — it is a precondition for claims to exist.

## Self-Evidence Test
**PASS** — Denying this axiom makes reasoning itself impossible.

## Independence Test
**PASS** — Cannot be derived from other axioms. It is the precondition for all other axioms.

## Consistency Test
**PASS** — Contradicts nothing; it is compatible with any consistent axiom set.

## Sufficiency Contribution
Enables reference — the ability to talk about the same thing twice.

## Alternative
No coherent alternative exists. Non-identity logics (paraconsistent systems) still preserve identity at the meta-level.

## Dependencies
None. This is the most foundational axiom.

## Evaluation Notes
Confidence 1.0 is reserved for axioms whose denial is not merely unlikely but literally incoherent.

## Lineage
Classical logic. Aristotle, Metaphysics IV.
