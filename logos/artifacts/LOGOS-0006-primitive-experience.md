# Primitive: Experience

## Statement
**Experience** is a primitive term. It cannot be defined without circularity — any attempt to define it relies on concepts that themselves presuppose experience (such as "awareness," "feeling," "perception," "consciousness").

## Justification for Primitive Status
- Defining experience as "what it is like to be something" (Nagel) uses "what it is like" — which is itself experiential.
- Defining experience as "subjective awareness" uses "subjective" and "awareness" — both of which presuppose experience.
- Every attempted definition either uses synonyms or relies on ostension ("you know what I mean").
- This is a genuine primitive in the Wittgensteinian sense: it can be shown but not said.

## Dependencies
None (primitive — foundational by definition)

## Evaluation Notes
This primitive is load-bearing for discussions of consciousness, AI alignment (can AI have experiences?), and moral status (does experience confer moral consideration?).
